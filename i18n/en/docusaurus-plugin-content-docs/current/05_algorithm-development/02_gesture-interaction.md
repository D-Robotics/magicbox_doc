# Gesture Interaction

This feature integrates D-Robotics gesture recognition algorithm to enable interaction with Magicbox through hand gestures.  
- For more details, refer to: [Gesture Recognition](https://developer.d-robotics.cc/rdk_doc/en/Robot_development/boxs/body/hand_gesture_detection)  
- Gesture control code can be found at: https://github.com/D-Robotics/magicbox_gesture_interaction
